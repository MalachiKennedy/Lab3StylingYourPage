By: Malachi Kennedy

CSS is an abbreviation for Cascading Style Sheets. There are 2 types of CSS. There are External and Internal CSS. Internal CSS is inside the HTML, located within the head section of an HTML document. It applies to the entire document, allowing for consistent styling across multiple elements. External CSS is outside the HTML. It is an external source stored in a separate file and can be linked to multiple HTML documents. You need to use (link rel="stylesheet" type="text/css" href="style.css"). You can use CSS to alter how your webpage looks. You can add color to your code. For example,"p{background-color: steelblue;}". You can change the color of the font. For example, "p{color:white;}".
